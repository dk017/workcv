# Homepage Wireframe

Last updated: June 11, 2026

Page: `workcv.co.uk/`

Purpose:

- blueprint for layout and section flow before implementation
- aligned to the homepage copy and visual design system
- Dwight-inspired structure, adapted for a UK CV builder

## Desktop layout summary

1. Sticky header
2. Hero split section
3. Trust strip
4. Problem / solution comparison block
5. How it works block
6. Template preview block
7. UK fit block
8. FAQ block
9. Final CTA block
10. Footer

## Mobile layout summary

Same section order, but all sections stack vertically.

Hero becomes:

- eyebrow
- H1
- subheading
- bullets
- primary CTA
- secondary CTA
- product preview

## 1. Sticky Header

### Layout

- left: logo `WorkCV`
- right nav: `Templates`, `Pricing`, `FAQ`
- right-side secondary CTA: `See example`
- right-side primary CTA: `Build my CV`

### Notes

- keep it light and compact
- white or warm translucent background
- sticky on scroll
- thin border or subtle shadow only when scrolled

## 2. Hero Section

### Layout

Desktop:

- left column: copy
- right column: product preview / CV builder mockup

Mobile:

- copy first
- visual second

### Left column content

- eyebrow
- H1
- subheading
- 3 bullets
- CTA row

### Right column content

- builder preview or CV mockup
- should look clean and practical, not overloaded

### Spacing notes

- generous vertical space
- copy should fit in one quick scan
- CTA row should sit immediately below bullets

## 3. Trust Strip

### Layout

- full-width band below hero
- 4 horizontal items on desktop
- stacked or marquee-like wrap on mobile

### Content blocks

- `Build free`
- `Pay GBP 4.99 when you download`
- `No subscription`
- `No automatic renewal`

### Notes

- simple, repeated reassurance
- no icons required unless they stay subtle

## 4. Problem / Solution Comparison Section

### Layout

- section label
- H2
- short intro paragraph
- 2-column comparison card
- CTA below card

Desktop:

- left column: `Typical CV builder`
- right column: `WorkCV`

Mobile:

- stacked cards, WorkCV second

### Left column content

- low trial price
- recurring renewal
- cancellation required
- broader platform than many users need

### Right column content

- build free
- pay GBP 4.99 once
- no subscription
- focused on finishing one strong CV

### Notes

- this is one of the most important sections on the page
- give it strong visual clarity and plenty of breathing room

## 5. How It Works Section

### Layout

- section label
- H2
- 3-step row on desktop
- stacked 3-step column on mobile

### Step blocks

1. Fill in your details
2. Choose a clean UK-ready template
3. Pay GBP 4.99 and download when ready

### Notes

- each step should be short
- icon optional
- do not let this become overly explanatory

## 6. Template Preview Section

### Layout

- section label
- H2
- short paragraph
- preview cards or one main preview plus smaller previews
- CTA below or beside previews

Desktop options:

- 3-card row
- or one large preview + two smaller supporting cards

Mobile:

- stacked cards

### Notes

- previews should feel practical
- no giant template gallery on homepage
- the goal is to show quality, not exhaust options

## 7. UK Fit Section

### Layout

- section label
- H2
- paragraph left
- support points right

Alternative:

- H2 and paragraph top
- 4-point grid below

### Support points

- no photo-first assumptions
- no date of birth or nationality in the standard CV flow
- practical structure for different experience levels
- clear PDF output for applications

### Notes

- this section is about local relevance, not SEO stuffing

## 8. FAQ Section

### Layout

- section label
- H2
- accordion list

### FAQ item count

- 5 items on homepage

### Notes

- keep answers short
- spacing should make the section feel calm, not dense

## 9. Final CTA Section

### Layout

- full-width CTA band
- centered content
- H2
- short supporting sentence
- CTA row

### CTA row

- primary: `Build my CV`
- secondary: `See pricing`

### Notes

- strong visual contrast from prior sections
- no excess copy here

## 10. Footer

### Layout

- logo / short positioning line
- link columns
- legal links

### Suggested link groups

- Templates
- Pricing
- FAQ
- Legal

## Responsive behavior

### Tablet

- keep hero two-column if it still breathes well
- comparison section may switch to stacked layout earlier than desktop

### Mobile

- reduce horizontal clutter
- stack all dual-column sections
- keep CTA buttons large enough to tap easily
- keep comparison content short enough to scan without fatigue

## Implementation priorities

If implementation time gets tight, preserve these sections first:

1. Hero
2. Trust strip
3. Problem / solution comparison
4. Final CTA

Those four sections carry the core commercial story.
